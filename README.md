# 🏠 Casa Inteligente con Riego Automático

## 📌 Descripción
Proyecto de automatización que permite controlar una casa inteligente con riego automático y luces desde el celular usando Blynk.

## ⚙️ Tecnologías utilizadas
- ESP32
- Sensor de humedad
- Módulo relé
- Blynk IoT

## 📱 Funciones
- Riego automático según humedad del suelo  
- Control de luces desde el celular  
- Conexión WiFi  

## 👥 Integrantes
- Tu nombre
- Compañeros

## 🔧 Objetivo
Optimizar el uso de agua y energía mediante automatización.
